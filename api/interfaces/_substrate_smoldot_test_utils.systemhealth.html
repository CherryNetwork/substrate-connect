Interface SystemHealth

SystemHealth is the type of the object in the result field of the JSON RPC responses from smoldot for the "system_health" RPC request.

remarks

We define SystemHealth instead of using the Health type in interfaces/system/types in @polkadot/api because its more ergonomic to use: primitive booleans and numbers instead of the polkadotjs wrapped Codec implementations of those.
