Class ConnectionManager

ConnectionManager is the main class involved in managing connections from apps and smoldots. It keeps track of apps in AppMediator instances and smoldot clients in SmoldotMediator instances. It is also responsible for triggering events when the state changes for the UI to update accordingly.

The 3 classes act in concert to multiples requests from various apps to smoldot clients and to clean up all an app's subscriptions when disconnected.
